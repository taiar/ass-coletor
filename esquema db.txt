Esquema do Bancod e Dados:

ass_tickers:

    id          - ID
    ticker      - código 
    razao       - nome da parada
    link        - apenas por segurança
    
ass_logs:

    id          - ID
    ticker_id   - ID do ticker correspondente
    data        - data Unixtime desta parcial
    valor       - valor (float) no momento
    
ass_fechamentos:

    id                      - ID
    ticker_id               - ID do ticker correspondente
    data                    - data do fechamento
    fechamento              - ultimo valor da acao no dia
    abertura                - valor com o qual a acao abriu o dia
    minimo                  - valor minimo durante o dia
    maximo                  - valor maximo durante o dia
    volume_financeiro (?)   - (?)
    variacao                - variacao = fechamento - abertura do dia
